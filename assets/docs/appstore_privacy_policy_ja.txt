プライバシーポリシー

最終更新日: 2026年1月4日

本プライバシーポリシーは、「Receipt App」（以下「本アプリ」）における個人情報の取扱いについて定めるものです。


第1条（事業者情報）

サービス名: ReceiptQR
運営者: 業務改善屋さん
所在地: 北海道帯広市東3条南12丁目1-5-205
連絡先: y_akagi@improve-biz.com


第2条（収集する情報）

本アプリは、サービス提供のために以下の情報を収集します：

1. ユーザー登録情報
   - メールアドレス
   - パスワード（暗号化して保存）
   - ユーザーID（自動生成）

2. 店舗情報
   - 店舗名
   - 店舗住所
   - 電話番号
   - インボイス番号
   - 印鑑画像（任意）

3. 領収書データ
   - 領収書番号
   - 発行日
   - 宛名
   - 金額情報
   - 但し書き
   - QRコード
   - PDFファイル

4. サブスクリプション情報
   - プラン種別（月額/年額）
   - 購入日時
   - 有効期限
   - トライアル状態


第3条（情報の利用目的）

収集した情報は、以下の目的で利用します：

1. サービスの提供・運営
2. 領収書の作成・管理
3. PDFの生成・保存
4. サブスクリプション管理
5. ユーザーサポート
6. サービスの改善・開発
7. 不正利用の防止


第4条（第三者サービスの利用）

本アプリは、以下の第三者サービスを利用しています：

1. Firebase（提供: Google LLC）
   - Firebase Authentication: ユーザー認証
   - Cloud Firestore: データベース
   - Firebase Storage: ファイルストレージ
   - プライバシーポリシー: https://policies.google.com/privacy

2. RevenueCat（提供: RevenueCat, Inc.）
   - サブスクリプション管理
   - 購入情報の処理
   - プライバシーポリシー: https://www.revenuecat.com/privacy

3. Apple App Store（提供: Apple Inc.）
   - アプリ内課金処理
   - プライバシーポリシー: https://www.apple.com/legal/privacy/


第5条（情報の保存期間）

1. ユーザーデータ: アカウント削除まで
2. 領収書データ: ユーザーが削除するまで（ソフト削除後90日で完全削除）
3. サブスクリプション情報: 税法に基づき7年間保存
4. ログデータ: 最大90日間


第6条（情報の共有）

当社は、以下の場合を除き、収集した情報を第三者に提供しません：

1. ユーザーの同意がある場合
2. 法令に基づく場合
3. 人の生命、身体または財産の保護に必要な場合
4. サービス提供に必要な範囲で業務委託先に提供する場合


第7条（セキュリティ）

当社は、収集した情報の漏洩、滅失、毀損を防止するため、以下の対策を講じています：

1. データの暗号化通信（TLS/SSL）
2. Firebase Security Rulesによるアクセス制御
3. パスワードのハッシュ化
4. 定期的なセキュリティ監査


第8条（ユーザーの権利）

ユーザーは、以下の権利を有します：

1. 自己の情報へのアクセス権
2. 情報の訂正・更新を求める権利
3. 情報の削除を求める権利（アカウント削除）
4. 情報処理の制限を求める権利
5. データポータビリティの権利
6. 異議を申し立てる権利

これらの権利を行使する場合は、アプリ内「設定」→「お問い合わせ」からご連絡ください。


第9条（Cookie・トラッキング技術）

本アプリは、Cookieやその他のトラッキング技術を使用していません。


第10条（子どものプライバシー）

本アプリは、13歳未満の子どもを対象としておらず、13歳未満の子どもから意図的に個人情報を収集することはありません。


第11条（国際データ転送）

本アプリは、Firebaseなどのクラウドサービスを利用しているため、収集した情報が日本国外のサーバーに保存される可能性があります。これらのサービスは、適切なデータ保護基準を満たしています。


第12条（プライバシーポリシーの変更）

当社は、必要に応じて本プライバシーポリシーを変更することがあります。変更後のプライバシーポリシーは、アプリ内での掲載をもって効力を生じます。


第13条（お問い合わせ）

本プライバシーポリシーに関するお問い合わせは、アプリ内「設定」→「お問い合わせ」からご連絡ください。
